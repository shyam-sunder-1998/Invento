About Invento

This is a fully voice controlled web based e-mail application which do not involves usage of any peripheral devices such as keyboard, mouse, etc. The user can compose mail, send it to the desired recipient, can check inbox, sent mail and recover deleted messages from trash. The application uses text-to-speech and speech-to-text functionalities.

The project eases the user convenience from the current mailing system as only our human voice is required to operate the product.
